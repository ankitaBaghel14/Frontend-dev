7. Install VS Code and Configure for HTML, CSS, JS:


Download VS Code from https://code.visualstudio.com/
 and install it.

Open VS Code, then install extensions:

Live Server → Auto-refresh pages on code changes.

HTML CSS Support → Helps with code suggestions and auto-completion.

JavaScript (ES6) Snippets → Quick JS coding.



Create a folder for your project, then add files:

index.html

style.css

script.js

Right-click index.html → Open with Live Server to view the page in browser.
(Take a screenshot of VS Code with your project open and Live Server running.)
